Project 3: Market analysis and Prediction in Minecraft using an Ensemble Model

April 9, 2017

  • One of my largest challenges so far! There is a Server on Minecraft that has an extremely active market on it, they use a bazaar trading system as well as an auction house system to create streams for players to sell & buy items. It’s become so popular that they created a public API that anyone can make calls to, you can get all kinds of data from them Check out their api here

  • Using Python I collected data for the entire summer and used a few discord servers full of players. I asked them to rate how “reliable” each item was as far as investing into it to turn a profit later on. The ratings range from 0-9 and were surprisingly not very uneven. Certain items always got rated poorly which is to be expected, but very rarely did one item in particular reign supreme!

  • After the collection of Data, I tried a few different approaches, a very simple approach was of course to make a random forest classifier and it performed fairly well, but I knew I could do better. And thus began the Creation of an ensemble of ML algorithms (SKLearn only) that helped push the predictions to another level! The models were weighted fairly evenly and simply casted a vote on what their prediction was.

  • The current model has a successful prediction rate of ~96% and has become so good, that people have begun using the code to improve components on their websites, as well as begin the process of in-game automation using a mixture of Baritone and the model above they are now making Sizeable and more importantly Stable profit without playing the game!

  • SKlearn, Random Forest Classification, Bagging, ADA Boosting, Gradient Boosting (non LGBM, purely sklearn), Extra Trees, KNN

  • Pandas, pandas is always useful.

  • GridsearchCV to help me determine the best final model without mindless guessing.

  • requests Used to make calls to the API, when gathering data to firstly create the dataset and finally to make predictions on current data from the game.
